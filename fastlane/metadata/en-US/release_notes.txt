What's new in v2.3.0:

New: Full-Page Compass Tool
- Beautiful full-screen compass accessible from Your Journey menu
- Real-time magnetic heading with smooth animated compass rose
- Bearing display in degrees with cardinal direction

New: Custom North Reference (Polaris Mode)
- Set any GPS coordinate as your "North" reference point
- Name and save multiple custom references for different hunts
- Real-time bearing and distance to your custom North target
- Perfect for treasure hunts that use landmark-based navigation

New: Saved Locations
- Bookmark any search result for instant offline access
- Saved locations appear first in all search results
- Manage favorites from Your Journey menu
- Reduces API usage and works without internet

New: Place Search in Compass
- Search for cities and places by name when setting custom North
- Type "Polaris, MT" to find and select locations quickly

Your data stays on your device. We don't track where you explore.
